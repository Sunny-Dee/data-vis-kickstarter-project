We can see that there is an interesting relationship going on for each city with an R square value of roughly 85%. Using the R package Rserve, we were able to create this model and save the fitted values in Tableau as a calculated field. From there, surplus (the Amount Pledged - Goal) was benchmarked against the expected surplus (fitted Amount Pledged- Goal). The results are what you see below.

What is interesting in this visualization is that while these cities and categories are the top performers in backings, we can plainly see how each country and category did better or worse than the overall expected trend. For example, while “Video Games” is one of the top performing categories, it grossly underperforms relative to how much we would normally expect it to.
